# SHCPhoto

  图片流浪、图片选择
  
  
  ### 功能
  
  
  ### 使用
  - 图片选择
  ```
  vc: 控制器
  maxSelected: 图片选择的最大值
  call: 图片回调
  SHCImagePickerViewController.show(vc: UIViewController, maxSelected: Int, call:@escaping ((_ images:[UIImage]?)->()))
  ```
  
  - 图片流浪
  ```
  vc: 控制器
  maxSelected: 图片选择的最大值
  call: 图片回调
  SHCImagePickerViewController.show(vc: UIViewController, maxSelected: Int, call:@escaping ((_ images:[UIImage]?)->()))
  ```
